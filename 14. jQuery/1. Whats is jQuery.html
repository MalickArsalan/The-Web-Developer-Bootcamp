What is jQuery?

jQuery is DOM Manipulation library

It helps us do things like selecting elements and manipulatiing elements, changing styles, adding event handlers and ajax.

It coes with a bunch of useful methods to things like:

  • Select Elements
  • manipulate Elements
  • Create Elements
  • Add Event Listeners
  • Aniate Elements
  • Add Effects
  • Make HTTP Requests(AJAX)

What is a library

The concept of a library in JavaScript is actually one that exists in on other programming language as well

The core idea is that we can take code that someone else wrote and we can add it to our own project.
That is what jQuery is.
